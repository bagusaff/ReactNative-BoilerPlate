# React Native Boiler Plate

Bare React Native Boiler Plate code. ( Last Updated 25 Feb 2022 )

## What's Inside ?

- Absolute file import ( dont need ../../.. to import files )
- React Navigation : Auth StackNavigator, BottomNavigator ( Home & Profile )
- Splash Screen
- Redux ready to use ( With Redux-Thunk & Redux-Persist )
- React Native dotenv
- Axios installed


## Installation

Clone this [repository](https://github.com/bagusaff/ReactNative-BoilerPlate.git).

```bash
git clone https://github.com/bagusaff/ReactNative-BoilerPlate.git
```

Install package
```bash
npm install
```

Run the program
```bash
npx react-native run-android
```
